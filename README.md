## Описание проекта
Приложение позволяет пользователям задавать вопросы к базе данных **на естественном языке** (русском или английском), а искусственный интеллект (локальная модель Ollama) автоматически генерирует соответствующий SQL-запрос.

**Основные возможности:**
- Генерация SQL-запросов с помощью LLM
- Поддержка русского и английского языков
- Выполнение запросов и отображение результатов в таблице
- История запросов
- Избранное (сохранение полезных запросов)
- Экспорт результатов в CSV и Excel
- Просмотр структуры базы данных
- Современный удобный веб-интерфейс

## Требования

- **Python 3.14+**
- **Ollama** (для работы локальной языковой модели, у меня используется llama3:8b )
- Установленные библиотеки: `flask`, `requests` (устанавливаются автоматически при первом запуске)

## Рекомендуемые модели Ollama
- `llama3:8b` / `phi3` / `mistral` — для лучшего качества
- `tinyllama` — для быстрого запуска (но качество ниже)

## Установка и запуск

### 1. Установка зависимостей

pip install flask requests

## 2. Создание демонстрационной базы данных

python database.py

## 3. Запуск Ollama
Перейти по ссылке: https://ollama.com/download
Скачать и установить OllamaSetup.exe

Открой отдельный терминал и выполни:

ollama serve
ollama pull tinyllama
ollama pull llama3:8b

## 4. Запуск приложения

python app.py

После запуска приложение автоматически откроется в браузере по адресу:
http://localhost:5000

## Как работать с приложением

- Перейдите во вкладку "Структура" — можно посмотреть таблицы и их поля.
- Введите вопрос на русском или английском (например: «Покажи 5 самых дорогих товаров»).
- Нажмите "Сгенерировать SQL".
- Проверьте и выполните запрос.
- Сохраняйте полезные запросы в Избранное.
